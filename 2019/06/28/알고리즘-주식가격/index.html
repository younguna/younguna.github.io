    알고리즘(Python) - 주식가격

    Input

    Expected Output

    목표

    • 초 단위로 기록된 주식가격이 담긴 배열 prices가 매개변수로 주어질 때, 가격이 떨어지지 않은 기간은 몇 초인지를 return 하도록 solution 함수를 완성하세요.

    조건

    • 1 <= prices.elements <= 10,000
    • 2 <= prices.length <= 100,000

    코드

    Python

    해설

    • input 배열의 길이를 저장합니다.
    • input 배열과 같은 길이의 0을 원소로 가지는 배열을 생성합니다.
    • input 배열을 하나씩 탐색하는 for 루프를 시작합니다.
    • 첫 번째 루프의 i의 총 변화를 탐색하는 j 루프를 시작합니다.
    • 만약 input 배열의 j번째 원소가 i번째 원소보다 작아지는 즉 가격이 하락하는 구간이 발생하면 그즉시 j 루프를 중단하고 그렇지 않다면 i가격의 하락이 발생하지 않은 것이니 i번째 0배열 원소를 1 증가시킵니다.
