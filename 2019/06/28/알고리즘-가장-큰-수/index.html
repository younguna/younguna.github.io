알고리즘 - 가장 큰 수

2019-06-28

알고리즘(Python, JavaScript) - 가장 큰 수

Input

Expected Output

목표

  • 입력 받은 배열의 숫자들을 이어 붙여 만들 수 있는 가장 큰 수를 리턴합니다.

조건

  • 1 <= numbers.length <= 100,000
  • 0 <= numbers.elements <= 1,000
  • 리턴 값의 타입은 문자열로 합니다.

코드

Python

해설

  • 각 원소들을 문자열로 바꾸어줍니다.
  • 원소들을 정렬하는데 파이썬의 문자열인 숫자의 정렬은 각 자릿 수를 기준으로 정렬을 진행합니다.
  • 또한 정렬의 기준을 똑같은 문자열을 세번 붙여놓은 것을 기준으로 하여 “3”과 “30”의 경우 “333”과 “303030”을 기준으로 정렬하게 되는데 333은 두번째 자리가 3인것에 반해 303030은 두번째 자리가 0이므로 333이 더 큰 것으로 판단됩니다.
  • 정렬된 배열을 하나의 문자열로 바꾸어 줍니다.
